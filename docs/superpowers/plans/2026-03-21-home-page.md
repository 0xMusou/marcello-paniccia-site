# Home Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the complete home page (`index.html`) with all 7 sections and GSAP/Lenis animations, establish the global scaffold shared by all future pages, and push everything to a new GitHub repository.

**Architecture:** Vanilla HTML/CSS/JS. Global styles (`css/global.css`) and global JS (`js/global.js`) are shared across all pages. Per-page files are isolated. Lenis is wired into GSAP's ticker so ScrollTrigger reads smooth-scrolled positions, not native scroll events — this is critical and must be set up before any ScrollTrigger animation is declared.

**Tech Stack:** HTML5, CSS custom properties, GSAP 3 + ScrollTrigger (CDN), Lenis v1 (CDN), self-hosted OTF fonts (Switzer Regular/Bold/Black + Gambarino Regular), GitHub CLI (`gh`)

---

| File | Responsibility |

- [ ] **Step 3: Create `js/global.js`** — Lenis init, GSAP ticker wiring, nav scroll behavior

```javascript
// ============================================================
// LENIS SMOOTH SCROLL + GSAP INTEGRATION
// ============================================================
const lenis = new Lenis({
  duration: 1.2,
  easing: (t) => Math.min(1, 1.001 - Math.pow(2, -10 * t)),
  orientation: 'vertical',
  smoothWheel: true,
});

// Wire Lenis into GSAP ticker — ScrollTrigger must read
// smooth-scrolled position, not native scroll position
lenis.on('scroll', ScrollTrigger.update);
gsap.ticker.add((time) => { lenis.raf(time * 1000); });
gsap.ticker.lagSmoothing(0);

// ============================================================
// NAV SCROLL BEHAVIOR
// ============================================================
const nav = document.querySelector('.nav');
lenis.on('scroll', ({ scroll }) => {
  nav.classList.toggle('scrolled', scroll > 60);
});
```

```javascript
// ============================================================
// HERO ENTRANCE ANIMATION
// ============================================================
(function initHero() {
  const tl = gsap.timeline({ defaults: { ease: 'power3.out' } });

  tl.to('.hero__overline',
      { opacity: 1, y: 0, duration: 0.7, delay: 0.2 })
    .fromTo('.hero__name-line[data-line="1"]',
      { opacity: 0, y: 60 },
      { opacity: 1, y: 0, duration: 0.9 },
      '-=0.3')
    .fromTo('.hero__name-line[data-line="2"]',
      { opacity: 0, y: 60 },
      { opacity: 1, y: 0, duration: 0.9 },
      '-=0.65')
    .to('.hero__tagline', { opacity: 1, y: 0, duration: 0.7 }, '-=0.4')
    .to('.hero__ctas',    { opacity: 1, y: 0, duration: 0.6 }, '-=0.4');

  // Scroll button scrolls to marquee section
  document.querySelector('.hero__scroll-btn')
    ?.addEventListener('click', () => lenis.scrollTo('.marquee'));
})();
```

## Implementation Notes

1. **Lenis + ScrollTrigger wiring is critical** — `lenis.on('scroll', ScrollTrigger.update)` and `gsap.ticker.add(...)` in `global.js` must be set up before any `ScrollTrigger` animation is defined. If omitted, animations fire at wrong scroll positions.

2. **`gsap.registerPlugin(ScrollTrigger)`** — the inline `<script>` tag in `index.html` registers this before `global.js` and `home.js` load. Do not move it after those script tags.

3. **CSS `:has()` for split hover** — supported in Chrome 105+, Firefox 121+, Safari 15.4+. No fallback needed for this project.

4. **File paths with spaces** — `optimized marcello photos/` contains a space. This is fine in HTML `src` attributes. Avoid using this path inside CSS `url()` — if a background-image is ever needed, use `%20` for the space.

5. **OTF font paths** — `global.css` uses `../OTF/` (one level up from `css/`). If directory structure changes, update all `@font-face` `src` paths.

6. **`gh` CLI** — Task 1 Step 7 uses `gh repo create`. If `gh` is not installed/authenticated, create the repo manually on github.com and use `git remote add origin` + `git push -u origin master`.
